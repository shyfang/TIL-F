- 类型转换
转换为 string 类型、转换为 number 类型和转换为 boolean 类型
// alert string
// number
undefined	NaN
null	0
true 和 false	1 and 0
string	去掉首尾空白字符（空格、换行符 \n、制表符 \t 等）后的纯数字字符串中含有的数字。如果剩余字符串为空，则转换结果为 0。否则，将会从剩余字符串中“读取”数字。当类型转换出现 error 时返回 NaN。
